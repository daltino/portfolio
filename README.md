# Portfolio — AI & Signal Processing Engineer

A personal portfolio and technical blog built with pure HTML/CSS/JS. Designed for deployment on GitHub Pages.

## 🚀 Deploy to GitHub Pages

1. Create a new GitHub repository (e.g. `yourusername.github.io` or `portfolio`)
2. Upload all files maintaining the folder structure:
   ```
   /
   ├── index.html
   ├── README.md
   ├── assets/
   │   └── style.css
   └── posts/
       ├── ai-software-development.html
       ├── signal-processing-ml.html
       ├── deploying-ml-production.html
       ├── time-series-deep-learning.html
       ├── feature-engineering-signals.html
       ├── ai-code-review.html
       ├── scalable-ml-architecture.html
       ├── research-to-production.html
       ├── real-time-audio-ml.html
       └── african-ai-opportunity.html
3. Go to **Settings → Pages → Source → main branch → / (root)**
4. Your site will be live at `https://yourusername.github.io`

## ✏️ Customization

- Replace `Dalton Whyte` in all HTML files with your actual name
- Replace `dalton.b.whyte@gmail.com` with your email
- Update GitHub and LinkedIn links in `index.html` footer
- Update the hero bio in `index.html` with your specific details
- Add your own posts by copying any post template in `/posts`
